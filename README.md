# Tichi Automation Testing

## QA Intern Assessment

This project is developed as part of the QA Intern Assessment.

## Project Overview

This project automates the login functionality of the Tichi Web Application using Selenium WebDriver with Python and the Page Object Model (POM) design pattern.

## Tech Stack

- Python
- Selenium WebDriver
- Pytest
- WebDriver Manager
- Chrome Browser

## Project Structure


## Automated Test Scenario

- Positive Login
- Valid Email
- Valid Password
- Successful Login

## Features

- Page Object Model (POM)
- Explicit Waits
- Pytest Framework
- ChromeDriver Manager
- Easy to Maintain

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the test:

```bash
python -m pytest -v
```
